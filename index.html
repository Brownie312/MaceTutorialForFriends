Written Guides

Wind Charge + Mace

Jump, then wind charge immediately to gain height. The higher you are, the more damage the mace does (especially with the right enchantments).


Keybinds

Use whatever feels comfortable. I remap 6–9 to R, T, F, C. RT is perfect for Wind Charge + Ender Pearl.


Wind Charge + Ender Pearl + Wind Charge

Wind charge upward → ender pearl → optional elytra → chestplate → mace before hitting the ground. This can instantly pop a totem.


Totem of Undying

Always keep a totem in your off-hand. Shields also work well against mace hits if timed correctly.


Screenshots & Tips

Installation Guides

Resource Packs

1. Download your chosen pack from the Resource Packs tab.

2. Open Minecraft and go to Options → Resource Packs → Open Pack Folder.

3. Drag and drop the downloaded .zip file into that folder.

4. In Minecraft, click the arrow icon on the pack to move it to "Selected".

Shader Packs

1. Install Iris Shaders (link available in the Shader Packs tab).

2. Download your shader pack (.zip file).

3. Go to Options → Video Settings → Shaders → Open Shader Folder.

4. Move the .zip file into the folder and select it in the game menu.

Mods & Fabric

1. Ensure you have the correct version of Minecraft installed.

2. Install the Fabric Loader (link below).

3. Download your mod (.jar file).

4. Go to Mods → Open Mods Folder and drop the .jar file inside.
